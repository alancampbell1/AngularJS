AngularJS Tutorial:
AngularJS was first released in 2012 by a Google employee. 
It is a JavaScript framework that can be added to a HTML page using a <script> tag.
A framework is a library written in JavaScript.
It extends HTML attributes using Directives and binds data to HTML using Expressions.

It is basically a distributed JavaScript file that can be added to a webpage using the
following script tag:

<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.9/angular.min.js"></script>
